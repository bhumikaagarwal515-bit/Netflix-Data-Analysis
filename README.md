# 🎬 Netflix Data Analysis using Python

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Titles dataset using Python. The goal is to explore trends in Netflix content, including movies, TV shows, genres, ratings, countries, and release years.

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 📂 Dataset

Netflix Titles Dataset

---

## 📊 Project Features

✔ Movies vs TV Shows Analysis

✔ Top 10 Genres

✔ Top 10 Countries

✔ Content Rating Distribution

✔ Release Year Analysis

✔ Correlation Heatmap

✔ Histogram Visualization

✔ Drama Content Analysis
